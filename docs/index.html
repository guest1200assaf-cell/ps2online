كيف تبدأ؟

01

حمّل وثبّت التطبيق

شغّل ملف Setup.exe واتبع خطوات التثبيت — يأخذ أقل من دقيقة

02

أضف ألعابك

اضغط "إضافة ألعاب" واختر ملفات ISO أو امسح مجلد الألعاب كامل

03

ثبّت المحاكي

من تبويب الإعدادات، اضغط "تثبيت PCSX2" — التطبيق يحمله من GitHub تلقائياً

04

العب مع صديقك

الـ Host يضغط "استضافة" ويشارك الكود، الـ Guest يضغط "انضمام" — وبس!
